# pythonls

`ls` alternative, inspired by [colorls](https://github.com/athityakumar/colorls)

 - Show directory `tree`
 - Sort by name or file type
 - Colored output with icons
 - Fully customizable (see `config.py`)
 - Under 150 lines of code
